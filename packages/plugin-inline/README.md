# JSX&thinsp;email Inline Plugin

`@jsx-email/plugin-inline` is a core JSX email plugin that converts CSS classes to email-safe inline styles.

This plugin is a peer dependency of `jsx-email` and is loaded automatically when the related render option, `inlineCss`, is enabled. If your package manager is configured not to automatically install peer dependencies, install this package manually before using that render option.

Please see the [documentation for this plugin](https://jsx.email/docs/plugins/inline) for more information.
