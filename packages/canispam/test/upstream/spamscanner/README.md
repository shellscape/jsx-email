# Spamscanner Test Attribution

These tests are converted from the public spamscanner test intent for browser-safe
comparison against `canispam`.

- Upstream source: https://github.com/spamscanner/spamscanner/tree/master/test
- Reviewed upstream branch: `master`
- Reviewed upstream package version: `6.1.5`
- Upstream license: Business Source License 1.1

Only scenarios relevant to generated JSX Email `.eml` content are run as
supported tests. Server-only scenarios are represented as skipped tests so the
capability boundary stays explicit.
