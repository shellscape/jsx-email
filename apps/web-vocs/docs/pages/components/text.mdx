---

## Install

:::code-group
```sh [pnpm]
pnpm add jsx-email
```

```sh [bun]
bun add jsx-email
```

```sh [npm]
npm add jsx-email
```

```sh [yarn]
yarn add jsx-email
```
:::

## Usage

```jsx "Text"
// TextExample.jsx
import { Text } from 'jsx-email';

const Email = () => {
  return <Text>Lorem ipsum</Text>;
};
```

## Props

### Optional Props

#### disableDefaultStyle

```ts
disableDefaultStyle?: boolean;
```

When `true`, removes default styles. Useful when using [`Tailwind`](/components/tailwind) or custom classes.

### Additional Props

This component expresses all of the [Common Component Props](https://react.dev/reference/react-dom/components/common) for `ComponentProps<'p'>`.
