## Install

:::code-group
```sh [pnpm]
pnpm add jsx-email
```

```sh [bun]
bun add jsx-email
```

```sh [npm]
npm add jsx-email
```

```sh [yarn]
yarn add jsx-email
```
:::

## Usage

```jsx "Html"
// HtmlExample.jsx
import { Body, Head, Html } from 'jsx-email';

const Email = () => {
  return (
    <Html lang="en" dir="ltr">
      <Head>
        <title>Email Title</title>
      </Head>
      <Body>Email content</Body>
    </Html>
  );
};
```

:::note[VML Support]
By default, includes VML namespace attributes for Outlook compatibility.
:::

## Props

### Optional Props

#### dir

```ts
dir?: string;
```

Text direction for email content.

#### lang

```ts
lang?: string;
```

Language identifier for email content.

#### enableVML

```ts
enableVML?: boolean;
```

Controls VML namespace attributes. Defaults to `true`.

### Additional Props

This component expresses all of the [Common Component Props](https://react.dev/reference/react-dom/components/common) for `ComponentProps<'html'>`.
