title: Hr

type: component
---

## Install

:::code-group
```sh [pnpm]
pnpm add jsx-email
```

```sh [bun]
bun add jsx-email
```

```sh [npm]
npm add jsx-email
```

```sh [yarn]
yarn add jsx-email
```
:::

## Usage

```jsx "Hr"
// HrExample.jsx
import { Hr } from 'jsx-email';

const Email = () => {
  return (
    <>
      <p>Content above</p>
      <Hr />
      <p>Content below</p>
    </>
  );
};
```

## Props

### Optional Props

#### disableDefaultStyle

```ts
disableDefaultStyle?: boolean;
```

When `true`, removes default styles. Useful when using [`Tailwind`](/components/tailwind) or custom classes.

### Additional Props

This component expresses all of the [Common Component Props](https://react.dev/reference/react-dom/components/common) for `ComponentProps<'hr'>`.
