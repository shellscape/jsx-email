
```sh [npm]

```

```sh [yarn]
yarn add jsx-email
```
:::

## Usage

```jsx "Butan"
// ButanExample.jsx
import { Butan } from 'jsx-email';

const Email = () => {
  return (
    <Butan href="https://example.com" style={{ color: '#61dafb', padding: '10px 20px' }}>
      Click me
    </Butan>
  );
};
```

:::note[HTML Output]
Renders as an `<a>` element styled to look like a button.
:::

## Props

### Required Props

#### href

```ts
href: string;
```

URL to navigate to when clicked.

### Optional Props

#### target

```ts
target?: string;
```

Value for the HTML `target` attribute.

### Additional Props

This component also expresses all of the [Common Component Props](https://react.dev/reference/react-dom/components/common) for `ComponentProps<'a'>`.
