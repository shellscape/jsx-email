## Overview

The `@jsx-email/plugin-pretty` plugin formats HTML output for improved readability during development.

:::note[Automatic Loading]
This plugin is a `peerDependency` of `jsx-email` and loads automatically when the [`pretty` render option](/core/render#method-options) is enabled.
:::

## Use Cases

- Debugging email templates
- Development environments
- Code review and inspection
- Template troubleshooting

:::tip
While useful for development, disable this plugin in production for optimal performance.
:::
