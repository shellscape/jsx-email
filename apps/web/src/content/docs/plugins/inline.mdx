## Overview

The `@jsx-email/plugin-inline` plugin converts CSS classes to inline styles, ensuring consistent styling across email clients.

:::note[Automatic Loading]
This plugin is a `peerDependency` of `jsx-email` and loads automatically when the [`inline` render option](/core/render#method-options) is enabled.
:::

## How It Works

Converts class-based styles to inline styles:

```html title="Before"
<style>
  .batman {
    border: 1px solid black;
  }
</style>
<div class="batman">batcave</div>
```

```html title="After"
<div style="border: 1px solid black">batcave</div>
```

## Performance Considerations

:::caution[Size Impact]
Inlined CSS increases HTML size significantly. Monitor console output for size warnings to avoid email clipping in clients like Gmail.
:::
