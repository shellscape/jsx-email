## Overview

The `@jsx-email/plugin-minify` plugin reduces HTML size by removing unnecessary whitespace and formatting.

:::note[Automatic Loading]
This plugin is a `peerDependency` of `jsx-email` and loads automatically when the [`minify` render option](/core/render#method-options) is enabled.
:::

## Benefits

- Reduces email size
- Prevents email clipping in clients
- Improves delivery performance

:::caution[Size Monitoring]
Monitor console output during rendering for size warnings to avoid email clipping in clients like Gmail.
:::
