## Usage

```jsx "Column"
// ColumnExample.jsx
import { Column, Row } from 'jsx-email';

const Email = () => {
  return (
    <Row>
      <Column>A</Column>
      <Column>B</Column>
      <Column>C</Column>
    </Row>
  );
};
```

:::note[HTML Output]
Renders as a `<td>` element within an email table structure.
:::

## Props

### TypeScript Interface

```ts
export interface ColumnProps extends BaseProps<'td'> {
  bgColor?: string;
  bgImage?: string;
}
```

### Optional Props

#### bgColor

```ts
bgColor?: string;
```

Sets the background color using HTML email's `bgcolor` property.

#### bgImage

```ts
bgImage?: string;
```

Sets the background image using HTML email's `background` property.

### Additional Props

This component expresses all of the [Common Component Props](https://react.dev/reference/react-dom/components/common) for `ComponentProps<'td'>`.
