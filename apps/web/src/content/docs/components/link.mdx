## Props

### Required Props

#### href

```ts
href: string;
```

URL for the link.

### Optional Props

#### disableDefaultStyle

```ts
disableDefaultStyle?: boolean;
```

When `true`, removes default styles. Useful when using [`Tailwind`](/components/tailwind) or custom classes.

#### target

```ts
target?: string;
```

Link target attribute (e.g., '\_blank').

### Additional Props

This component expresses all of the [Common Component Props](https://react.dev/reference/react-dom/components/common) for `ComponentProps<'a'>`.
