# Samples App

<div>
  <img src="https://raw.githubusercontent.com/shellscape/jsx-email/main/assets/brackets.svg" alt="JSX email" valign="sub">
  A project for building and testing the JSX email samples preview
  <br/>
<div>

The intent of this app is to provide likely real-world templates for `jsx-email` and its preview command. These templates are also used to build the deployed samples site at `https://jsx.email/samples`.

## Requirements

This package requires an [LTS](https://github.com/nodejs/Release) Node version (v22.0.0+) and React v19.1.0+.

## Testing

To begin, you'll need to be setup to work within this repo. Please see _Getting Started_ in the [CONTRIBUTING.md](../../CONTRIBUTING.md) guide before continuing.

To start the preview server locally, run:

```console
$ moon run samples:dev
```

Open up the preview server [http://localhost:55420/](http://localhost:55420/)
