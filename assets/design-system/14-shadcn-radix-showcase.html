Radix → shadcn replacement showcase

Standalone CDN demo of shadcn-equivalent components for the primitives in scope (Select, Popover, Toggle Group, Collapsible, Dialog, Tabs, Separator, and Slot guidance).

CDN imports used: https://cdn.tailwindcss.com and https://cdn.jsdelivr.net/npm/motion@11.11.13/+esm.

Effect notes

  • CSS / data-state Select and Popover open/close animations, Collapsible height transitions, Dialog overlay/content transitions, and Tabs active-state visuals.
  • Motion-driven Toggle Group active-pill movement uses Motion (spring animation) as a CDN-friendly analogue of the framer-motion layout animation pattern seen on main.

Slot note (shadcn composition)

For Slot, shadcn does not replace Radix; it intentionally reuses @radix-ui/react-slot for asChild polymorphism. 

import { Slot } from "@radix-ui/react-slot"

function Button({ asChild, ...props }) {
  const Comp = asChild ? Slot : "button"
  return <Comp {...props} />
}
